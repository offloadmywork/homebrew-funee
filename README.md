# Offloadmywork Funee

## How do I install these formulae?

`brew install offloadmywork/funee/<formula>`

Or `brew tap offloadmywork/funee` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "offloadmywork/funee"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
